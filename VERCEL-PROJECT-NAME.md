# Vercel 프로젝트 이름 설정

## 문제
프로젝트 이름이 대문자를 포함하거나 잘못된 형식입니다.

## 해결 방법

### 프로젝트 이름 규칙
- ✅ 소문자만 사용 (a-z)
- ✅ 숫자 사용 가능 (0-9)
- ✅ 특수문자: `.`, `_`, `-` 사용 가능
- ❌ 대문자 사용 불가
- ❌ `---` (하이픈 3개 연속) 불가

### 올바른 프로젝트 이름 예시
- ✅ `kucn`
- ✅ `kucn-app`
- ✅ `kucn_app`
- ✅ `kucn-app-2024`
- ✅ `kucnapp`

### 잘못된 프로젝트 이름 예시
- ❌ `KUCN` (대문자)
- ❌ `KUCN-App` (대문자 포함)
- ❌ `kucn---app` (하이픈 3개 연속)
- ❌ `kucn app` (공백)

## 설정 방법

Vercel 프로젝트 설정 화면에서:

1. **Project Name** 필드 찾기
2. **소문자로 변경**: `kucn` 또는 `kucn-app` 입력
3. **Continue** 또는 **Deploy** 클릭

## 추천 프로젝트 이름

가장 간단한 이름:
```
kucn
```

또는:
```
kucn-app
```
